# 📝 Paste Management App

A modern and responsive **Paste Management Application** built with **React**, **Redux Toolkit**, and **Tailwind CSS**. The application allows users to create, edit, delete, search, and manage text snippets efficiently. All pastes are stored in **Browser Local Storage**, ensuring data persists even after refreshing or closing the browser.

## ✨ Key Features

- **Full CRUD Functionality:** Users can seamlessly **Create, Read, Update (Edit), and Delete** pastes.
- **Persistent Data Storage:** Integrated with **Browser Local Storage** to ensure user data remains safe and saved even after refreshing or closing the browser.
- **Global State Management:** Uses **Redux Toolkit (Slices)** to handle global data flow cleanly across different components and pages.
- **Real-time Search & Filter:** Quickly filter through saved pastes using a dynamic search feature.
- **Clean UI & Iconography:** Built with a modern responsive interface using **Tailwind CSS** and **Lucide React Icons**.

## ⚙️ Installation & Setup

Clone the repository

```bash
git clone https://github.com/sabirkhandev/paste-app.git
```

Go to the project directory

```bash
cd paste-app
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```
